# Hackathon Platform

##  Overview
A full-stack Hackathon Management System where users can explore hackathons, create/join teams, and collaborate. Admin can manage hackathons and teams.

---

##  Features
- User Authentication (Login/Register)
- Role-based Access (Admin/User)
- Create & Delete Hackathons (Admin)
- Team Creation & Join Requests
- Accept / Reject Join Requests
- Blog Section
- About Page
- Logout Functionality

---

##  Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: MySQL

---

##  Project Structure
frontend → React UI  
backend → Spring Boot APIs
